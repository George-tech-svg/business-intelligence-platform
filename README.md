# Business Intelligence Platform 
 
## What This Project Does 
This is a complete data platform that collects data from multiple sources, cleans it, stores it in a MySQL database, runs machine learning predictions, and displays everything on a dashboard.

## How to Install 
1. Install Python 3.9 or higher

3. Set up MySQL database
